Bienvenidos Libreta Virtual

Es una aplicación web completa diseñada para facilitar la gestión integral de las faltas, amonestaciones y seguimientos de los alumnos en una institución educativa.

Ofrece una plataforma centralizada donde directivos, docentes y padres pueden acceder a la información relevante y actualizada sobre el comportamiento académico y disciplinario de cada estudiante.
